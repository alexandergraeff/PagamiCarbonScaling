Estimating Biomoss Loss from Pagami Creek Fire

Alexander Graeff

2026-02-25

Goal

Summarize biomass data per forest type for reference and burn plots: calculate an average biomass per forest type and then take the difference between reference plots and burns plots within a forest type to estimate the amount of biomass lost from the burn.

  }

Format Data

Here we pare down the biomass table and join the reference tables, then select columns relevant to the analyses. This is saved in data/processed as it is the data frame used to perform downstream analyses.

  data <- biomass.raw %>% 
    select(1, 4) %>%  # just take the total biomass and plot ID columns
    rename(tpIdent = 1, biomassTotal = 2) %>%  # rename them
    left_join(forestTypes, join_by(tpIdent)) %>%  # join the forest types data

Calulate Biomass Differences in Groups

First, we will calculate by subtracting the average reference plot biomass within each forest type from each individual burn plot in that forest type. Then we can average the burn plots within each forest type.

We can calculate this a couple of different ways, but we should end up with roughly the same answer.

An intermediate step in this code allows us to see the estimated biomass lost individually for each plot. Otherwise, the calculations are probably more convoluted than the next step (Calculation B). But, it is more flexible as we will want to summarize by different groupings and may want to calculate averages directly, or leave plots with individual values (for use in boxplots).

  biomass_diff_a <- function(data, x, y = NULL, sev_col = NULL, calculation) {  # define function with required x, optional y and sev_col
    sev_quo <- enquo(sev_col)  # capture sev_col as quosure (tidy-eval)
    has_sev <- !quo_is_null(sev_quo)  # flag: TRUE if sev_col was supplied
  
    a <- data %>%
      select(tpIdent, biomassTotal, forestType,  # keep needed columns
             forestType_group, ref_burn, crnSev) %>%
      group_by({{x}}) %>%  # group by primary variable x
        mutate(biomassMean_ref = mean(biomassTotal[ref_burn == "reference"])) %>%  # calculate mean biomass of reference plots within x
        ungroup() %>%  # remove grouping
      group_by({{x}}, {{y}}) %>%  # group by primary variable x and secondary variable y (if given)
        mutate(biomassDiff = if_else(condition = ref_burn == "burn",  # for burn rows, calculate biomassDiff...
                                     true = biomassMean_ref - biomassTotal,  # as ref mean biomass minus individual plot biomass
                                     false = NA)) %>%
        ungroup() # remove grouping
  
        if (calculation == "average") { a %>%  # if we want to calculate the calculation means across groups then
              filter(ref_burn == "burn") %>%  # keep only burn rows
              { 
                if (has_sev) filter(., !is.na(!!sev_quo)) else .  # and if we're including crown severity then remove plots which don't have a crown severity
                } %>%  # optionally require non-missing sev_col
              group_by({{x}}, {{y}}) %>%  # group by x and y
                summarize(meanDiff = mean(biomassDiff), .groups = "keep") %>%  # compute mean difference per group (consider na.rm = TRUE)
                ungroup() %>%  # remove grouping
              arrange({{x}}, {{y}})
          }  # order by x then y
        else if (calculation == "unaveraged") { a %>%  # otherwise, we won't perform any calculations...
          filter(ref_burn != "reference") %>%  #... but we will remove all reference plots and...
          {
          if (has_sev) filter(., #... if we are summarizing by crown burn severity... 
                              !is.na(!!sev_quo))  #... then we will remove any plots which don't have a severity value
          else .
          }
        }
        else if (calculation == "intermediate") {
          a
        }
      } # end function

Example of Calculation A output.

  data %>% 
    biomass_diff_a(x = forestType, 
                   y = NULL,

If we average biomass for burn and reference plots, then subtract, it should be just about the same as the calculation above.

This code is probably more intuitive, (and is much simpler) but doesn’t include the separation of reference calculation so we can’t easily add crown severity as a grouping variable.

Indeed we get the same values, which is a good check, but we want the more flexible calculation going forward - so we’ll just use Calculation A.

  biomass_diff_b <- function(data, ...) {
    data %>% 
      group_by(...) %>% 
        summarize(biomassMean = mean(biomassTotal),
                  .groups = "keep") %>% 
        ungroup() %>% 
      pivot_wider(names_from = ref_burn, 
                  names_glue = "{ref_burn}_biomass",
                  values_from = biomassMean, 
                  values_fill = 0) %>% 
      mutate(meanBiomassDiff = reference_biomass - burn_biomass)
  }

For visualization, we summarize the data for plotting using the biomass_diff_a function.

By Forest Type

Column Chart

# generate a table to label number of plots ----

  nLabs <- data %>% 
    n_labs(forestType, ref_burn) %>%

# perform calculations ----

  pData <- data %>% 
    biomass_diff_a(x = forestType, 
                   y = NULL,

  pData
# plot ----
  
  pData %>% 
    ggplot(mapping = aes(x = forestType, y = meanDiff, fill = forestType)) +
      geom_col() +
      theme_minimal() +
      theme(axis.title.x = element_blank(),

      geom_hline(yintercept = 0) +
      labs(y = "Average Biomass Loss (mg/ha)",
